Прототип сайта в редакторе Figma

Прототип сайта для интернет-магазина цветов "Мир цветов".Разработан в программе figma.com. Состоит из 3-х страниц, содержащих формы регистрации и обратной связи. Между страницами и формами выполнены связи.

Калькулятор на JavaScript

Калькулятор выполняет основные арифметические операции, такие как сложение, вычитание, умножение и деление. Имеет кнопку для очистки экрана и кнопку Backspace, имеет функцию памяти, оформлен с помощью CSS, реализован с помощью чистого и читабельного JavaScript-кода.

Приложение Список дел

Приложение создано с помощью VUE.JS. Возможность добавления новых пунктов. Возможность удаления из списка. Функция сортировки и фильтрации списка. Оформлено с помощью CSS. Реализовано с помощью чистого и читабельного JavaScript-кода.
